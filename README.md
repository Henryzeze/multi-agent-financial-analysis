# 🧠 Multi-Agent Financial & Data Agents

## 📌 Overview
This project implements a **multi-agent agentic system** designed to perform financial analysis and Data insight analysis.

The system leverages **LLM-powered agents** to collaboratively perform tasks such as planning, reasoning, and execution in a coordinated workflow.

## 🧰 Technologies Used

- Python  
- CrewAI  
- LangChain 
- OpenAI API / LLMs
